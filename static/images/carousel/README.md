# Imágenes del Carousel

Esta carpeta contiene las imágenes para el carousel de la página principal.

## Imágenes requeridas:
- biblioteca-1.jpg (1200x500px) - Biblioteca principal
- biblioteca-2.jpg (1200x500px) - Sala de lectura
- biblioteca-3.jpg (1200x500px) - Recursos digitales
- estudiantes-1.jpg (1200x500px) - Estudiantes estudiando

## Formato recomendado:
- Resolución: 1200x500 píxeles
- Formato: JPG o PNG
- Tamaño máximo: 500KB por imagen
- Calidad: Alta para mejor visualización
